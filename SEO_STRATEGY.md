## PART 5: On-Page SEO Checklist

### Every Page Must Have:

#### Title Tag
- Format: `[Primary Keyword] in [City] | Aadhirai`
- Length: 50–60 characters
- Examples:
  - `Pharmacy Billing Software in Chennai | Medora+ by Aadhirai`
  - `ERP Software for Small Business | Aadhirai Innovations`

#### Meta Description
- 150–160 characters
- Include keyword + unique value prop + CTA hint
- Example: `Best pharmacy billing software in Chennai. GST-compliant, offline-first, expiry tracking. Free trial + quick setup. Trusted by 50+ pharmacies.`

#### H1 (One per page)
- Must include primary keyword
- Natural, not stuffed
- Example: `Pharmacy Billing Software in Chennai for Medical Shops and Chains`

#### H2, H3 Structure
- H1 (main heading)
  - H2 (major section)
    - H3 (subsection)
    - H3 (subsection)
  - H2 (major section)

#### Keyword Placement
- H1: Primary keyword
- First paragraph: Keyword within first 100 words
- H2s: Include variations (not exact match stuffing)
- Meta description: Primary keyword
- URL slug: Primary keyword (if short/clean)
- Alt text: One or two images with keyword

#### Internal Linking
- 5–8 internal links per page
- Use descriptive anchor text (not "click here")
- Link to:
  - Related products
  - City pages
  - Blog articles
  - Main product page

#### Image Optimization
- File name: relevant keyword (e.g., `pharmacy-billing-software-screenshot.jpg`)
- Alt text: `Screenshot of Medora+ pharmacy billing software showing GST invoicing`
- Size: Compress to <100KB (use TinyPNG)

#### Schema Markup
- Minimum: LocalBusiness + Product + FAQ
- See section 6 below

## PART 6: Technical SEO

### 6.1 Sitemap Generation

**Automated sitemap (Express.js example):**

```javascript
const products = ['pharmacy-billing', 'erp-software', 'pos-system'];
const cities = ['chennai', 'madurai', 'coimbatore', 'salem', ...];
const urls = [];

products.forEach(product => {
  // Main product page
  urls.push(`https://aadhiraiinnovations.com/products/${product}`);
  
  // City pages
  cities.forEach(city => {
    urls.push(`https://aadhiraiinnovations.com/${product}/${city}`);
  });
});

// Generate sitemap.xml
```

**Sitemap structure:**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://aadhiraiinnovations.com/pharmacy-billing-software</loc>
    <changefreq>monthly</changefreq>
    <priority>0.9</priority>
  </url>
  <url>
    <loc>https://aadhiraiinnovations.com/pharmacy-billing-software/chennai</loc>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
  ...
</urlset>
```

## PART 15: Common Mistakes to Avoid

🚫 **Keyword Stuffing**
- Don't repeat "pharmacy billing software" 10 times per page
- Natural keyword density: 0.5–1% is fine

🚫 **Thin Content on City Pages**
- Each city page should be 700+ words minimum
- Unique, localized content (not template clones)

🚫 **Ignoring Mobile**
- 60%+ of SaaS searches happen on mobile
- Ensure fast, responsive, easy-to-tap pages

🚫 **Weak CTAs**
- "Click here" or "Learn more" don't convert
- Use action-oriented copy: "Talk to us", "See a demo", "Start free trial"

🚫 **Slow Site Speed**
- >3 second load = 40% bounce rate increase
- Compress images, minify code, use CDN

🚫 **Broken Internal Links**
- Regularly audit for 404s and redirect chains
- Use GSC to find crawl errors

🚫 **No Schema Markup**
- Miss out on featured snippets, rich results
- LocalBusiness + Product + FAQ are minimum

🚫 **Ignoring Competitors**
- Research what Zoho, Tally, other systems rank for
- Find content gaps you can fill
