1. Copy this folder to `C:\xampp\htdocs\aadhirai-genic`
2. Open `http://localhost/aadhirai-genic/index.html`

## Email sending
In `assets/js/app.js` enable one method in `sendLead()`:
- Formspree: paste your endpoint & uncomment
- EmailJS: include SDK & add keys
- Serverless: implement `/api/contact`

## Replace images
Swap any file in `/assets/img/` with same name to use real photos/screens.
